phpMyKey

A simple program that requires a one-time-use key to unlock some content.

$ cd some_folder
$ git clone https://github.com/KevZho/phpmykey.git

Features

  • Admin control panel for running cron, adding and removing keys.
  • Single use keys with customized content.
  • Easy to use interface.
